# Interview-Prepration 
This repository contains a comprehensive collection of **Data Structures and Algorithms (DSA) questions** curated for **technical interview preparation**.
It is designed to help developers strengthen problem-solving skills and prepare for coding interviews in product-based and service-based companies.
---

## 🚀 Purpose

* To cover **all major DSA topics**
* To provide **interview-focused questions**
* To build **strong problem-solving skills**
* To help in **cracking coding rounds (6–20 LPA roles)**

---

## 📂 Topics Covered

### 🔹 Arrays

* Basic operations
* Sliding window
* Prefix sum
* Kadane’s Algorithm

### 🔹 Strings

* String manipulation
* Pattern matching
* Anagrams & Palindromes

### 🔹 Linked List

* Singly & Doubly Linked List
* Reversal techniques
* Cycle detection

### 🔹 Stack & Queue

* Stack applications
* Monotonic stack
* Queue & Deque problems

### 🔹 Recursion & Backtracking

* Subsets & permutations
* N-Queens
* Sudoku solver

### 🔹 Searching & Sorting

* Binary Search (Important)
* Merge Sort, Quick Sort
* Search in rotated arrays

### 🔹 Trees

* Binary Trees
* BST operations
* Traversals (DFS, BFS)

### 🔹 Heaps / Priority Queue

* Min/Max Heap
* Top K problems
* Heap sort

### 🔹 Bit Manipulation

* XOR tricks
* Bit masking

---

## 💡 How to Use

1. Start with basic topics (Arrays, Strings)
2. Move to intermediate (Linked List, Stack, Queue)
3. Practice advanced topics (DP, Graphs)
4. Focus on:

   * Time Complexity ⏱️
   * Space Optimization 💾
   * Clean Code ✨

---
## 🎯 Interview Strategy

* Solve **easy → medium → hard**
* Practice **daily (2–5 problems)**
* Revise frequently 🔁
* Focus on **patterns, not just questions**
